# Snake Game

This is a classic snake game built with React, designed to run in a web browser.

## Features

*   Classic snake gameplay: control a snake to eat food and grow longer.
*   Game ends if the snake runs into itself or the walls.
*   Selectable difficulty levels:
    *   **Low**: The snake moves at a slow speed.
    *   **Medium**: The snake moves at a moderate speed.
    *   **High**: The snake moves at a fast speed.

## How to Play

1.  Use the arrow keys (Up, Down, Left, Right) to control the direction of the snake.
2.  Eat the food that appears on the screen to score points and make the snake grow.
3.  Avoid colliding with the walls or the snake's own body.
4.  The game gets progressively faster and more challenging.

## Development

This project was developed using [React](https://reactjs.org/).
